# Cover Novel

将小说隐藏在代码注释中阅读。

## 功能

- **代码伪装阅读** — 小说文本嵌入 JavaScript `//` 注释中，看起来像在写代码
- **老板键** — `Alt+Q` 快速隐藏/恢复小说，自动切回之前编辑的文件
- **翻页快捷键** — `Alt+Right` 下一页 / `Alt+Left` 上一页（支持自定义）
- **章节导航** — 自动识别 `第X章`/`第X回` 等标记，快速跳转到指定章节（精准定位到标题行）
- **TXT 编码自动检测** — 自动识别 GBK / UTF-8 / Big5 编码，无需手动转码
- **多书管理** — 从任意位置打开 `.txt` 文件，自动记录最近阅读列表，切换时高亮当前书
- **进度保存** — 自动保存阅读进度，下次打开从断点继续
- **状态栏导航** — 「上一页」「下一页」「跳页」「章节」按钮 + 进度显示

## 快捷键

| 快捷键 | 命令 | 说明 |
|--------|------|------|
| `Ctrl+Alt+,` / `Cmd+Alt+,` | 打开小说 | 快速打开 novel.js |
| `Alt+Right` | 下一页 | 翻到下一页 |
| `Alt+Left` | 上一页 | 翻到上一页 |
| `Alt+Q` | 老板键  | 隐藏/恢复小说 |

所有快捷键均可通过 VS Code 的 `keybindings.json` 自定义覆盖。

## 使用

1. 按 `Ctrl+Alt+,` 打开小说
   - 首次使用会弹出文件选择器，选一个 `.txt` 文件
   - 之后直接恢复上次的阅读进度
2. 使用状态栏按钮或快捷键翻页
3. 点状态栏右下角的进度可切换已打开的书籍
